# MVC-GossipProject
 L'utilisateur peut rentrer les ragots et rumeur, afficher une liste des potins, puis supprimer ceux qui ne sont plus d'actualité.

Les vues seront affichées dans le terminal, les potins enregistrés en CSV, et le programme tournera jusqu'à ce que l'utilisateur le quitte.

Lancez app.rb pour utiliser !
